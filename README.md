# Plant Disease Classifier

Классификатор болезней растений на основе сверточной нейронной сети ResNet18 с Telegram-ботом.

## Описание

Проект определяет болезни растений по фотографии листа. Модель обучена на датасете PlantVillage (38 классов болезней). Включает консольный скрипт для предсказаний и Telegram-бота для удобного взаимодействия.

## Возможности

- Классификация 38 видов болезней растений
- Transfer learning с использованием ResNet18
- Поддержка GPU для ускорения обучения
- Автоматическое сохранение лучшей модели
- Telegram-бот для получения предсказаний
- Работа с изображениями без сохранения на диск

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/fan1omass/plants_analysis
cd plant-doctor
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Подготовьте данные:
```bash
python src/prepare_data.py
```

4. Настройте переменные окружения:
```bash
# Создайте файл .env в корне проекта
echo "TOKEN=your_telegram_bot_token" > .env
```

## Использование

### Обучение модели

```bash
python src/train.py
```

Параметры обучения настраиваются в `src/train.py`:
- NUM_EPOCHS: 10
- LEARNING_RATE: 0.001
- BATCH_SIZE: 32

### Предсказание через консоль

```bash
python src/predict.py
```

Укажите путь к изображению в `predict.py`:
```python
IMAGE_PATH = Path("path/to/your/image.jpg")
```

### Запуск Telegram-бота

```bash
python src/app.py
```

Получите токен бота у @BotFather в Telegram и добавьте в `.env`.

## Ограничения

Модель обучена на датасете PlantVillage, который снимался в лабораторных условиях. Это влияет на точность при работе с реальными фотографиями.

Почему точность ниже на реальных фото:
- В датасете однородный фон, в реальности фон разнообразный
- В датасете студийное освещение, в реальности разное освещение
- В датасете лист в центре кадра, в реальности разный ракурс

Рекомендации для лучших результатов:
- Фотографируйте лист на однородном фоне
- Используйте хорошее освещение
- Заполняйте листом большую часть кадра

## Технические детали

Архитектура:
- Base: ResNet18 (предобученная на ImageNet)
- Замороженные слои: все кроме последнего полносвязного слоя
- Обучаемые параметры: около 19,000
- Входное изображение: 224x224 RGB

Аугментация данных:
- Случайный поворот (30 градусов)
- Горизонтальное и вертикальное отражение
- ColorJitter (яркость, контраст, насыщенность, тон)
- Normalization (ImageNet mean/std)

Оборудование:
- Рекомендуется GPU (CUDA)
- Протестировано на Google Colab T4
